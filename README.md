# Hova
### Very Tiny Programming Language.


## Features
- Declare variables
- Print Strings
- Print Variables
- Connect Variables
- Calculate Variables
- for loops
- while loops
- inputs
- functions
- if conditions


## Examples

Examples you will get in Documentation.md or on https://github.com/BennoCrafter/Hova/wiki
